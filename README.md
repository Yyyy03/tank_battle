# Tank Battle MVP

基于原生 HTML/CSS/JS + Canvas 的《坦克大战》MVP（单页面即可运行）。

## 运行方式

任选其一：

1. 直接双击打开 `index.html`。
2. 在项目根目录启动静态服务：

```bash
python -m http.server
```

然后在浏览器打开 `http://localhost:8000/`。

## 操作说明

- 移动：WASD 或方向键
- 发射：Space
- 重新开始：R

## 玩法说明（MVP）

- 玩家坦克生命值 3。
- 敌方坦克至少 3 辆，具备随机移动/转向/开火的简单 AI。
- 子弹可与墙体/坦克/边界发生碰撞。
- 地图为 20x15 网格，包含砖墙（可被击毁）与钢墙（不可击毁）。
- 击毁敌方坦克得分 +100。
- 左上角显示生命值与分数，Game Over 时显示覆盖层。
